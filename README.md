# Dungeon Inn Frontend

A dark, atmospheric massage reservation system built with Next.js.

## Theme
Dungeon Inn 🕯️ - A cozy, fire-lit tavern aesthetic with deep browns and amber accents.

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Redux Toolkit
- Material UI

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
